# JumpServer

SSL未対応のサーバーとのデータのやり取りを、SSL対応の別サーバーで中継するスクリプト

## 何ができる？

- SSL未対応サイトの疑似SSL導入

```txt
例:
http://12.34.56.78/
に格納してあるデータを
https://hogefuga.com/
というURLから接続
```

- ポートフォワーディング

```txt
例:
http://12.34.56.78:9012/
に格納してあるデータを
https://hogefuga.com/
というURLから接続
```

## 使い方

- このプロジェクトをクローンして、/\_\_target.phpに以下の内容を記述する

```php<?php
// ここを実際の転送先に変更してください
$targetDomain = "http://12.34.56.78:9012/";
```

- 疑似SSL接続用の、既にSSL化されているサーバーにこのプロジェクトを配置

```txt
例:
https://hogefuga.com
からアクセスする場合、そこにFTPとかでアップロード
```

- 完成！
